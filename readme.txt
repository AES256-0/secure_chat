About: A simple secure chat program that uses encryption for confedentiality of data.
       It is written in python 3.7 and tested on linux machine.
Requirements:
       *python3.7 need to be installed
       *key_file need to be shared manually
       *To install the requirements ,simply run the following command on the terminal
                                #chmod u+x requirement
                                #./requirement
How to run:
           #python securechatprogram.py --option client/keys/server --port port_no --ip ip_address
           where --option keys will generate new keys and copy to a file in your current directory.
           you need to copy this file to your other device whom you want to chat.
